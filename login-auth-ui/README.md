# Login Authentication UI

A responsive and user-friendly authentication frontend built with **React.js**. This application integrates with the Spring Boot Authentication API to provide a complete user authentication experience.

---

## Features

- User Registration
- User Login
- Forgot Password
- OTP Verification
- Reset Password
- React Router Navigation
- Axios API Integration
- Responsive UI
- Clean and Beginner-Friendly Code

---

## Tech Stack

### Frontend

- React.js
- React Router DOM
- Axios
- HTML5
- CSS3
- JavaScript (ES6)

## Getting Started

### Clone Repository

git clone https://github.com/<your-username>/login-auth-ui.git

### Open Project

cd login-auth-ui

### Install Dependencies

npm install

### Start Development Server


npm start

Application will start on:

http://localhost:3000
